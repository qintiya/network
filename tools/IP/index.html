AWDev IP Intelligence & Domain DNS Diagnostic Suite

Welcome to the AWDev Webmaster & Network Engine. Designed for network engineers, cybersecurity analysts, web developers, and system administrators, this suite offers a comprehensive set of real-time tools to analyze IP addresses, trace email headers, audit domain health, and monitor global DNS propagation.

Key Tools & Features

  • What is My IP Address: Instant detection of your public IPv4/IPv6 address, ISP details, and approximate geographical location.
  • IP Location Lookup & WHOIS: Determine ownership, autonomous system number (ASN), organization, and regional internet registry (RIR) data for any IP.
  • Trace Email Header Analyzer: Paste raw email headers to identify sender IP addresses, intermediate relay hops, and authentication flags (SPF, DKIM, DMARC).
  • IP & Email Blacklist Checker: Query global spam databases (DNSBL/RBL) to ensure your server IP or domain is not blacklisted.
  • Global DNS Checker & Health Audit: Verify DNS record propagation (A, AAAA, MX, NS, TXT, CNAME) across distributed servers worldwide and run comprehensive health checks for domain configuration.
